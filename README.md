# kickstart
The Kickstart project is a campaign kickstarter using Ethereum.
It includes a smart contract and a user interface.
One can:
- Create campaigns
- Contribute to campaigns
- Create reuqests for sending the money raised to recipients
- Approve requests
- Finalize requests

Technologies:
- Solidity
- React
